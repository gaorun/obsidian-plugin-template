# obsidian-plugin-template

Monorepo for Obsidian plugin development tooling.

## Packages

| Package              | Description                                                                 |
| -------------------- | --------------------------------------------------------------------------- |
| `packages/template/` | Obsidian plugin development template — start building your own plugin       |
| `packages/cli/`      | `@gaorun/create-obsidian-plugin` — scaffolding CLI to bootstrap new plugins |

## Getting started

```bash
pnpm install
pnpm build
```

## Requirements

- Node.js 24.13.0+
- pnpm 11.13.1+
